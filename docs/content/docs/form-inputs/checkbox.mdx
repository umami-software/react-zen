# Checkbox

A checkbox allows a user to select multiple items from a list of individual items, or to mark one individual item as selected.

<Example>
  <Checkbox>Accept terms and conditions.</Checkbox>

  ```jsx
  <Checkbox>Accept terms and conditions.</Checkbox>
  ```
</Example>

## Indeterminate

<Example>
  <Checkbox isIndeterminate={true}>Accept terms and conditions.</Checkbox>

  ```jsx
  <Checkbox isIndeterminate>Accept terms and conditions.</Checkbox>
  ```
</Example>

## Disabled

<Example>
  <Checkbox isDisabled={true}>Accept terms and conditions.</Checkbox>

  ```jsx
  <Checkbox isDisabled>Accept terms and conditions.</Checkbox>
  ```
</Example>

## Component props

  <DataTable
