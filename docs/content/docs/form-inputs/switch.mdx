# Switch

A switch allows a user to turn a setting on or off.

<Example>
  <Switch>Low power mode</Switch>

  ```jsx
  <Switch>Low power mode</Switch>
  ```
</Example>

## Disabled

<Example>
  <Switch isDisabled>Disabled off</Switch>
  <Switch defaultSelected={true} isDisabled>
    Disabled on
  </Switch>

  ```jsx
  <Switch isDisabled>Disabled off</Switch>
  <Switch defaultSelected isDisabled>Disabled on</Switch>
  ```
</Example>
