# ComboBox

A combo box combines a text input with a listbox, allowing users to filter a list of options to items matching a query.

<Example>
  <ComboBox
    items={['One', 'Two', 'Three', 'Four', 'Five', 'Six', 'Seven', 'Eight', 'Nine', 'Ten']}
  />

  ```jsx
  <ComboBox
    items={['One', 'Two', 'Three', 'Four', 'Five', 'Six', 'Seven', 'Eight', 'Nine', 'Ten']}
  />
  ```
</Example>
