# ComboBox

A combo box combines a text input with a listbox, allowing users to filter a list of options to items matching a query.

<Example>
  <ComboBox>
    <ListItem id="one">One</ListItem>
    <ListItem id="two">Two</ListItem>
    <ListItem id="three">Three</ListItem>
    <ListItem id="four">Four</ListItem>
    <ListItem id="five">Five</ListItem>
    <ListItem id="six">Six</ListItem>
    <ListItem id="seven">Seven</ListItem>
    <ListItem id="eight">Eight</ListItem>
    <ListItem id="nine">Nine</ListItem>
    <ListItem id="ten">Ten</ListItem>
  </ComboBox>

  ```jsx
  <ComboBox>
    <ListItem id="one">One</ListItem>
    <ListItem id="two">Two</ListItem>
    <ListItem id="three">Three</ListItem>
    <ListItem id="four">Four</ListItem>
    <ListItem id="five">Five</ListItem>
  </ComboBox>
  ```
</Example>
