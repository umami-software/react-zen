# Shadows

Shadows are a set of `box-shadow` values that give the element a sense of depth.

<Example gap="5">
  <Label>sm</Label>
  <ExampleBox shadow="sm" />
  <Label>md</Label>
  <ExampleBox shadow="md" />
  <Label>lg</Label>
  <ExampleBox shadow="lg" />
  <Label>xl</Label>
  <ExampleBox shadow="xl" />
  <Label>2xl</Label>
  <ExampleBox shadow="2xl" />

  ```jsx
  <Box shadow="sm">Small shadow</Box>
  <Box shadow="md">Medium shadow</Box>
  <Box shadow="lg">Large shadow</Box>
  <Box shadow="xl">Extra large shadow</Box>
  <Box shadow="2xl">2XL shadow</Box>
  ```
</Example>
