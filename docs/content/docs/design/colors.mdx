title: Colors
---

Our color system is based on Tailwind CSS colors. There are two categories of colors: semantic colors and accent colors.

## Semantic colors

Semantic colors provide meaning to UI elements and are used for common states.

<Example direction="row" gap="2">
  <Column alignItems="center">
    <ExampleBox backgroundColor="primary" />
    <Text size="xs">primary</Text>
  </Column>
  <Column alignItems="center">
    <ExampleBox backgroundColor="muted" />
    <Text size="xs">muted</Text>
  </Column>
  <Column alignItems="center">
    <ExampleBox backgroundColor="transparent" />
    <Text size="xs">transparent</Text>
  </Column>
</Example>

## Accent colors

Accent colors provide additional meaning to UI elements and are used in buttons, links and statuses.

</Example>

## Text colors

<Example direction="row" gap="4">
  <Text color="primary">Primary text</Text>

## Component props

<Example>
  <DataTable
    data={[
      { name: 'color', value: '"primary" | "muted" | "disabled" | AccentColor' },

## Available accent colors

The following Tailwind colors are available as accent colors:

- `gray`, `slate`, `zinc`, `neutral`, `stone`
- `red`, `orange`, `amber`, `yellow`, `lime`
- `green`, `emerald`, `teal`, `cyan`, `sky`
- `blue`, `indigo`, `violet`, `purple`, `fuchsia`
- `pink`, `rose`
