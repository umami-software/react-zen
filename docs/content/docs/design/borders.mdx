# Borders

Borders are always solid with a default border color. You can control the `position`, `radius`, and `color` of borders.

## Border position

<Example gap="5">
  <Label>Default</Label>
  <ExampleBox border backgroundColor={true} borderRadius="none" />
  <Label>Top</Label>
  <ExampleBox border="top" backgroundColor={true} borderRadius="none" />
  <Label>Right</Label>
  <ExampleBox border="right" backgroundColor={true} borderRadius="none" />
  <Label>Bottom</Label>
  <ExampleBox border="bottom" backgroundColor={true} borderRadius="none" />
  <Label>Left</Label>
  <ExampleBox border="left" backgroundColor={true} borderRadius="none" />

  ```jsx
  <Box border>Default</Box>
  <Box border="top">Top</Box>
  <Box border="right">Right</Box>
  <Box border="bottom">Bottom</Box>
  <Box border="left">Left</Box>
  ```
</Example>
