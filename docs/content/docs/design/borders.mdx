# Borders

Borders are always solid with a default border color. You can control the `position`, `radius`, and `color` of borders.

  <Label>Border right</Label>
  <ExampleBox border="right" backgroundColor={true} rounded="none" />
  <Label>Border bottom</Label>
  <ExampleBox border="bottom" backgroundColor={true} rounded="none" />
  <Label>Border left</Label>
  <ExampleBox border="left" backgroundColor={true} rounded="none" />
</Example>
