# Toast

A succinct message that is displayed temporarily.

<Example>
  <ToastExample />

  ```jsx
  <Button onPress={() => toast('Your message has been sent.')}>Show toast</Button>
  ```
</Example>

## With title

<Example>
  <ToastTitleExample />

  ```jsx
  <Button
    onPress={() =>
      toast('The application encountered an error.', { title: 'Oops! Something went wrong.' })
    }
  >
    Show toast
  </Button>
  ```
</Example>

## With action

<Example>
  <ToastActionExample />

  ```jsx
  <Button
    onPress={() =>
      toast('The task was deleted successfully.', {
        title: 'Task deleted',
        actions: ['Undo'],
      })
    }
  >
    Show toast
  </Button>
  ```
</Example>

## Variants

<Example>
  <ToastVariantsExample />

  ```jsx
  <>
    <Button onPress={() => toast('Your message was sent.', { variant: 'success' })}>
      Success toast
    </Button>
    <Button onPress={() => toast('Something went wrong.', { variant: 'error' })}>
      Error toast
    </Button>
  </>
  ```
</Example>
