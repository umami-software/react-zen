# Icon label

An element that displays an icon next to a label.

<Example>
  <Row>
    <IconLabel icon={<ExampleIcons.Settings />}>Settings</IconLabel>
  </Row>

  ```jsx
  <IconLabel icon={<SettingsIcon />}>Settings</IconLabel>
  ```
</Example>
