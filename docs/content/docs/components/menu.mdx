# Menu

A menu displays a list of actions or options that a user can choose.

<Example>
  <Menu>
    <MenuItem>One</MenuItem>
    <MenuItem>Two</MenuItem>
    <MenuItem>Three</MenuItem>
    <MenuItem>Four</MenuItem>
    <MenuItem>Five</MenuItem>
  </Menu>

  ```jsx
  <Menu>
    <MenuItem>One</MenuItem>
    <MenuItem>Two</MenuItem>
    <MenuItem>Three</MenuItem>
  </Menu>
  ```
</Example>

## Submenu

<Example>
  <Menu>
    <MenuItem id="one">One</MenuItem>
    <MenuItem id="two">Two</MenuItem>
    <MenuItem id="three">Three</MenuItem>
    <MenuItem id="four">Four</MenuItem>
    <SubMenuTrigger>
      <MenuItem id="five" showSubMenuIcon>Five</MenuItem>
      <Popover>
        <Menu>
          <MenuItem id="six">Six</MenuItem>
          <MenuItem id="seven">Seven</MenuItem>
          <MenuItem id="eight">Eight</MenuItem>
        </Menu>
      </Popover>
    </SubMenuTrigger>
  </Menu>

  ```jsx
  <Menu>
    <MenuItem>One</MenuItem>
    <SubMenuTrigger>
      <MenuItem showSubMenuIcon>More</MenuItem>
      <Popover>
        <Menu>
          <MenuItem>Sub item 1</MenuItem>
          <MenuItem>Sub item 2</MenuItem>
        </Menu>
      </Popover>
    </SubMenuTrigger>
  </Menu>
  ```
</Example>
