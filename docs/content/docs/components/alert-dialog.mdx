# Alert Dialog

Alert dialogs are a specific type of Dialog. They display important information that users need
to acknowledge.

<Example>
  <AlertDialogExample />

  ```jsx
  <DialogTrigger>
    <Button>Delete Account</Button>
    <AlertDialog
      title="Delete account?"
      description="This action cannot be undone."
      isDanger
      confirmLabel="Delete"
      onConfirm={() => handleDelete()}
    />
  </DialogTrigger>
  ```
</Example>
