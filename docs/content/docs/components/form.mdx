# Form

A form wrapper built on react-hook-form that provides validation, error handling, and field state management. Combines with FormField components to create accessible, validated forms with minimal boilerplate.

<Example>
  <Box width="300px">
    <FormExample />
  </Box>

  ```jsx
  <Form onSubmit={(data) => console.log(data)}>
    <FormField name="email" label="Email" rules={{ required: true }}>
      <TextField />
    </FormField>
    <FormSubmitButton>Submit</FormSubmitButton>
  </Form>
  ```
</Example>

## Array fields

<Example>
  <Box width="300px">
    <FormFieldArrayExample />
  </Box>

  ```jsx
  <Form defaultValues={{ items: ['Item 1'] }}>
    <FormFieldArray name="items">
      {({ fields, append, remove }) => (
        <>
          {fields.map((field, index) => (
            <FormField key={field.id} name={`items.${index}`}>
              <TextField />
            </FormField>
          ))}
          <Button onPress={() => append('')}>Add Item</Button>
        </>
      )}
    </FormFieldArray>
  </Form>
  ```
</Example>
