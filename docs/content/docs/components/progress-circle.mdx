# Progress Circle

A circular progress indicator that visually represents completion status. Ideal for compact spaces, dashboards, or when you want a more visually distinctive progress display than a bar.

<Example>
  <ProgressCircle value={45} />
</Example>

## Show value

<Example>
  <ProgressCircle value={15} showValue />
</Example>
