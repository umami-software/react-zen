# Progress Bar

A horizontal bar that visually represents the completion status of a task or process. Supports determinate progress with percentage values and optional value labels.

<Example>
  <ProgressBar value={50} />

  ```jsx
  <ProgressBar value={50} />
  ```
</Example>

## Show value

<Example>
  <ProgressBar value={25} showValue />

  ```jsx
  <ProgressBar value={25} showValue />
  ```
</Example>

## Component props

<Example>
  <DataTable
    data={[
      { name: 'value', value: 'number', description: 'Current progress value' },
      { name: 'minValue', value: 'number', description: 'Minimum value (default: 0)' },
      { name: 'maxValue', value: 'number', description: 'Maximum value (default: 100)' },
      { name: 'showPercentage', value: 'boolean', description: 'Shows percentage text' },
    ]}
  >
    <DataColumn id="name" label="Name" />
    <DataColumn id="value" label="Type" />
    <DataColumn id="description" label="Description" />
  </DataTable>
</Example>
