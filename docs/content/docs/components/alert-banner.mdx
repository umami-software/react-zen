# Alert Banner

Alert banners show pressing and high-signal messages, such as system alerts.
They're meant to be noticed and prompt users to take action.

<Example>
  <AlertBanner
    title="New update available!"
    description="Version 2.0 has been released."
    icon={<Lucide.Rocket />}
  >
    <Button>Dismiss</Button>
    <Button variant="primary">Download</Button>
  </AlertBanner>

  ```jsx
  <AlertBanner
    title="New update available!"
    description="Version 2.0 has been released."
    icon={<Lucide.Rocket />}
  >
    <Button>Dismiss</Button>
    <Button variant="primary">Download</Button>
  </AlertBanner>
  ```
</Example>

## With close button

<Example direction="column">
  <AlertBanner
    variant="info"
    title="Dismissible alert"
    description="Click the X to close this alert."
    allowClose
  />

  ```jsx
  <AlertBanner
    variant="info"
    title="Dismissible alert"
    description="Click the X to close this alert."
    allowClose
    onClose={() => setVisible(false)}
  />
  ```
</Example>
