---
title: Loading Button
---

A button that indicates an action is still pending.

<Example>
  <LoadingButton isLoading />
</Example>

## With text

<Example>
  <LoadingButton isLoading>Loading...</LoadingButton>
</Example>

## Variants

<Example>
  <LoadingButton variant="primary" isLoading>
    {'Loading...'}
  </LoadingButton>
  <LoadingButton isLoading>
    {'Loading...'}
  </LoadingButton>
  <LoadingButton variant="outline" isLoading>
    {'Loading...'}
  </LoadingButton>
  <LoadingButton variant="quiet" isLoading>
    {'Loading...'}
  </LoadingButton>
  <LoadingButton variant="danger" isLoading>
    {'Loading...'}
  </LoadingButton>
</Example>
