# Loading Button

A button that indicates an action is still pending.

<Example>
  <LoadingButton isLoading />

  ```jsx
  <LoadingButton isLoading />
  ```
</Example>

## With text

<Example>
  <LoadingButton isLoading>Loading...</LoadingButton>

  ```jsx
  <LoadingButton isLoading>Loading...</LoadingButton>
  ```
</Example>
