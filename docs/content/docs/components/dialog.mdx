# Dialog

A dialog is an overlay shown above other content in an application.

<Example>
  <DialogExample />

  ```jsx
  <DialogTrigger>
    <Button>Open Dialog</Button>
    <Modal>
      <Dialog title="Dialog Title">
        <Text>Dialog content goes here.</Text>
      </Dialog>
    </Modal>
  </DialogTrigger>
  ```
</Example>
