# Slider

A slider allows a user to select a value within a range.

<Example>
  <Slider label="Opacity" defaultValue={50} style={{ maxWidth: 300 }} />

  ```jsx
  <Slider label="Opacity" defaultValue={50} />
  ```
</Example>
