# Toggle

A toggle button allows a user to toggle a selection on or off, for example switching between two states or modes.

<Example>
  <Toggle>{`B`}</Toggle>
</Example>

## Component props

<Example>
  <DataTable
    data={[
      { name: 'label', value: 'string', description: 'Label text displayed above the toggle' },
      { name: 'isSelected', value: 'boolean', description: 'Controlled selected state' },
      { name: 'defaultSelected', value: 'boolean', description: 'Default selected state' },
      { name: 'isDisabled', value: 'boolean', description: 'Disables the toggle' },
      { name: 'onChange', value: '(isSelected: boolean) => void', description: 'Called when state changes' },
    ]}
  >
    <DataColumn id="name" label="Name" />
    <DataColumn id="value" label="Type" />
    <DataColumn id="description" label="Description" />
  </DataTable>
</Example>
