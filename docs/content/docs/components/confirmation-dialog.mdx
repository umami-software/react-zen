# Confirmation Dialog

A high-friction dialog for destructive or irreversible actions. Requires users to type a confirmation phrase before proceeding, preventing accidental deletions or critical changes.

<Example>
  <ConfirmationDialogExample />

  ```jsx
  <DialogTrigger>
    <Button variant="danger">Delete account</Button>
    <Modal>
      <ConfirmationDialog
        title="Delete account"
        value={'abc123'}
        confirmLabel="Delete"
        isDanger={true}
      >
        <Text>This cannot be undone. All your data will be permananently deleted.</Text>
      </ConfirmationDialog>
    </Modal>
  </DialogTrigger>
  ```
</Example>
