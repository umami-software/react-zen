# Confirmation Dialog

A high-friction dialog for destructive or irreversible actions. Requires users to type a confirmation phrase before proceeding, preventing accidental deletions or critical changes.

<Example>
  <ConfirmationDialogExample />

  ```jsx
  <DialogTrigger>
    <Button variant="danger">Delete Repository</Button>
    <ConfirmationDialog
      title="Delete repository?"
      value="my-repo"
      confirmMessage="Type 'my-repo' to confirm deletion."
      isDanger
      onConfirm={() => handleDelete()}
    />
  </DialogTrigger>
  ```
</Example>
