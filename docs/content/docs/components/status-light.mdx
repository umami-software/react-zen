# Status Light

Status lights are used to color code categories and labels commonly found in data visualization.

<Example>
  <StatusLight>Status</StatusLight>
</Example>

## Variants

<Example direction="column">
  <StatusLight variant="success">Success</StatusLight>
  <StatusLight variant="error">Error</StatusLight>
  <StatusLight variant="warning">Warning</StatusLight>
  <StatusLight variant="active">Active</StatusLight>
  <StatusLight variant="inactive">Inactive</StatusLight>
</Example>
