# Icon

A wrapper around SVG elements to help control size, color and rotation.

<Example>
  <Icon>
    <Lucide.Wifi />
  </Icon>

  ```jsx
  <Icon>
    <Lucide.Wifi />
  </Icon>
  ```
</Example>

## Fill and stroke colors

Control SVG fill and stroke colors using the `fillColor` and `strokeColor` props.

<Example direction="row" gap="4">
  <Icon size="lg" fillColor="blue-500">
    <Lucide.Heart />
  </Icon>
  <Icon size="lg" strokeColor="red-500" strokeWidth="2">
    <Lucide.Heart />
  </Icon>
  <Icon size="lg" fillColor="green-500" strokeColor="green-700" strokeWidth="1">
    <Lucide.Heart />
  </Icon>

  ```jsx
  <Icon size="lg" fillColor="blue-500">
    <Lucide.Heart />
  </Icon>
  <Icon size="lg" strokeColor="red-500" strokeWidth="2">
    <Lucide.Heart />
  </Icon>
  <Icon size="lg" fillColor="green-500" strokeColor="green-700" strokeWidth="1">
    <Lucide.Heart />
  </Icon>
  ```
</Example>
