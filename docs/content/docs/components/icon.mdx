# Icon

A wrapper around SVG elements to help control size, color and rotation.

<Example>
  <Icon>
    <ExampleIcons.Wifi />
  </Icon>

  ```jsx
  <Icon>
    <WifiIcon />
  </Icon>
  ```
</Example>

## Sizes

<Example>
  <Icon size="xs">
    <ExampleIcons.Wifi />
  </Icon>
  <Icon size="sm">
    <ExampleIcons.Wifi />
  </Icon>
  <Icon size="md">
    <ExampleIcons.Wifi />
  </Icon>
  <Icon size="lg">
    <ExampleIcons.Wifi />
  </Icon>
  <Icon size="xl">
    <ExampleIcons.Wifi />
  </Icon>

  ```jsx
  <Icon size="xs"><WifiIcon /></Icon>
  <Icon size="sm"><WifiIcon /></Icon>
  <Icon size="md"><WifiIcon /></Icon>
  <Icon size="lg"><WifiIcon /></Icon>
  <Icon size="xl"><WifiIcon /></Icon>
  ```
</Example>

## Rotation

<Example direction="row">
  <Icon size="md">
    <ExampleIcons.Wifi />
  </Icon>
  <Icon size="md" rotate={90}>
    <ExampleIcons.Wifi />
  </Icon>
  <Icon size="md" rotate={180}>
    <ExampleIcons.Wifi />
  </Icon>
  <Icon size="md" rotate={270}>
    <ExampleIcons.Wifi />
  </Icon>

  ```jsx
  <Icon rotate={90}><WifiIcon /></Icon>
  <Icon rotate={180}><WifiIcon /></Icon>
  <Icon rotate={270}><WifiIcon /></Icon>
  ```
</Example>
