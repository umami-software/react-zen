# Icon

A wrapper around SVG elements to help control size, color and rotation.

<Example>
  <Icon>
    <Lucide.Wifi />
  </Icon>

  ```jsx
  <Icon>
    <Lucide.Wifi />
  </Icon>
  ```
</Example>

## Tailwind SVG classes

The Icon component supports all Tailwind SVG utilities including `fill-*`, `stroke-*`, and `stroke-{width}` classes.

<Example direction="row" gap="4">
  <Icon size="lg" className="fill-blue-500">
    <Lucide.Heart />
  </Icon>
  <Icon size="lg" className="stroke-red-500 stroke-2">
    <Lucide.Heart />
  </Icon>
  <Icon size="lg" className="fill-green-500 stroke-green-700 stroke-1">
    <Lucide.Heart />
  </Icon>

  ```jsx
  <Icon size="lg" className="fill-blue-500">
    <Lucide.Heart />
  </Icon>
  <Icon size="lg" className="stroke-red-500 stroke-2">
    <Lucide.Heart />
  </Icon>
  <Icon size="lg" className="fill-green-500 stroke-green-700 stroke-1">
    <Lucide.Heart />
  </Icon>
  ```
</Example>
