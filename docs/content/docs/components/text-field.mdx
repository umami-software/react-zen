# Text Field

A text field allows a user to enter a plain text value with a keyboard.

<Example>
  <TextField />
</Example>

## Placeholder text

<Example>
  <TextField placeholder="Enter some text" />
</Example>

## Disabled

<Example>
  <TextField defaultValue="Disabled" isDisabled />
</Example>

## Read only

<Example>
  <TextField defaultValue="Read only" isReadOnly />
</Example>

## As textarea

<Example>
  <TextField placeholder="Enter some text." asTextArea />
</Example>

## Copy button

<Example>
  <TextField placeholder="Enter some text." value="Text" style={{ width: 200 }} allowCopy />
  <TextField
    placeholder="Enter some text."
    value="Text"
    style={{ width: 200 }}
    asTextArea
    allowCopy
  />
</Example>

## Quiet

<Example>
  <Column gap>
  <TextField placeholder="Enter some text." variant="quiet" style={{ fontSize: '2rem' }} />
  <TextField placeholder="Enter some text." variant="quiet" />
  </Column>
</Example>
