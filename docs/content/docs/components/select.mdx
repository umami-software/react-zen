# Select

Displays a list of options for the user to pick fromâ€”triggered by a button.

<Example>
  <Box width="300px">
    <Select items={['One', 'Two', 'Three', 'Four', 'Five']} placeholder="Please select an option" />
  </Box>
</Example>

## Fullscreen mode

<Example>
  <Box width="300px">
    <Select
      label="Select"
      items={['One', 'Two', 'Three', 'Four', 'Five']}
      placeholder="Please select an option"
      isFullscreen
    />
  </Box>
</Example>
