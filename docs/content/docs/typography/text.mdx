# Text

A text element with comprehensive typography controls including size, color, weight, spacing, and text decorations.

<Example>
  <Text>Here is some text.</Text>

  ```jsx
  <Text>Here is some text.</Text>
  ```
</Example>

## Vertical align

Control vertical alignment of inline elements. Useful for superscript, subscript, and aligning text with icons.

<Example>
  <Column gap="4">
    <Column gap="1">
      <Label>Superscript</Label>
      <Text size="lg">E = mc<Text verticalAlign="super" size="xs">2</Text></Text>
    </Column>
    <Column gap="1">
      <Label>Subscript</Label>
      <Text size="lg">H<Text verticalAlign="sub" size="xs">2</Text>O</Text>
    </Column>
    <Column gap="1">
      <Label>Top</Label>
      <Box border padding="2" borderRadius="md">
        <Text>
          <Box as="span" className="inline-block w-6 h-6 bg-blue-500 rounded mr-2" />
          <Text verticalAlign="top">aligned to top</Text>
        </Text>
      </Box>
    </Column>
    <Column gap="1">
      <Label>Middle</Label>
      <Box border padding="2" borderRadius="md">
        <Text>
          <Box as="span" className="inline-block w-6 h-6 bg-green-500 rounded mr-2" />
          <Text verticalAlign="middle">aligned to middle</Text>
        </Text>
      </Box>
    </Column>
    <Column gap="1">
      <Label>Bottom</Label>
      <Box border padding="2" borderRadius="md">
        <Text>
          <Box as="span" className="inline-block w-6 h-6 bg-orange-500 rounded mr-2" />
          <Text verticalAlign="bottom">aligned to bottom</Text>
        </Text>
      </Box>
    </Column>
  </Column>
</Example>
