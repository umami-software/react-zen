---
title: Text
---

A text element with controls for size, color, weight and letter spacing.

<Example>
  <Text>Here is some text.</Text>
</Example>

## Colors

<Example>
  <Text>Normal text</Text>
  <Text color="muted">Muted text</Text>
  <Text color="disabled">Disabled text</Text>
  <Text color="red">Colored text</Text>
</Example>

## Sizes

<Example>
  <Text size="xs">Size xs</Text>
  <Text size="sm">Size sm</Text>
  <Text size="base">Size base</Text>
  <Text size="lg">Size lg</Text>
  <Text size="xl">Size xl</Text>
  <Text size="2xl">Size 2xl</Text>
  <Text size="3xl">Size 3xl</Text>
  <Text size="4xl">Size 4xl</Text>
</Example>

## Weights

<Example>
  <Text weight="light">Light text</Text>
  <Text weight="normal">Normal text</Text>
  <Text weight="medium">Medium text</Text>
  <Text weight="semibold">Semibold text</Text>
  <Text weight="bold">Bold text</Text>
</Example>

## Alignment

<Example>
  <Box width="100%">
    <Text align="left" as="div">
      {'Left'}
    </Text>
    <Text align="center" as="div">
      {'Center'}
    </Text>
    <Text align="right" as="div">
      {'Right'}
    </Text>
  </Box>
</Example>

## Letter spacing

<Example>
  <Text spacing="tighter">Letter spacing tighter</Text>
  <Text spacing="tight">Letter spacing tight</Text>
  <Text spacing="normal">Letter spacing normal</Text>
  <Text spacing="wide">Letter spacing wide</Text>
  <Text spacing="wider">Letter spacing wider</Text>
</Example>

## Formatting

<Example>
  <Text italic>Italic</Text>
  <Text underline>Underline</Text>
  <Text strikethrough>Strikethrough</Text>
</Example>

## Transform

<Example>
  <Text transform="capitalize">Capitalize</Text>
  <Text transform="uppercase">Uppercase</Text>
  <Text transform="lowercase">Lowercase</Text>
</Example>
