# Code

An inline code element for displaying code snippets, file names, commands, or technical terms within text. Styled with a monospace font and subtle background for visual distinction.

<Example>
  <Text>
    You can make changes to your <Code>package.json</Code> file.
  </Text>

  ```jsx
  <Text>
    Run <Code>npm install</Code> to install dependencies.
  </Text>
  ```
</Example>

## Component props

<Example>
  <DataTable
    data={[
      { name: 'children', value: 'ReactNode', description: 'Code content' },
    ]}
  >
    <DataColumn id="name" label="Name" />
    <DataColumn id="value" label="Type" />
    <DataColumn id="description" label="Description" />
  </DataTable>
</Example>
