# Heading

A semantic heading component for titles and section headers. Supports multiple sizes and semantic HTML levels (h1-h6) for proper document structure and accessibility.

<Example>
  <Heading>Hello.</Heading>

  ```jsx
  <Heading>Hello.</Heading>
  ```
</Example>

## Component props

Heading extends [Text](/docs/typography/text) and accepts all Text props. Default values are `size="2xl"`, `weight="semibold"`, and `spacing="tight"`.
