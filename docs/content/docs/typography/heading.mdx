# Heading

A heading element.

<Example>
  <Heading>Hello.</Heading>
</Example>

## Alignment

<Example>
  <Box width="100%">
    <Heading align="left">Left</Heading>

## Component props

Heading extends [Text](/docs/typography/text) and accepts all Text props.

<Example>
  <DataTable
    data={[
      { name: 'size', value: 'Responsive<HeadingSize>', description: '"sm" | "base" | "lg" | "xl" | "2xl" | ... | "9xl"' },

      { name: 'spacing', value: 'Responsive<LetterSpacing>', description: '"tighter" | "tight" | "normal" | "wide" | "wider" | "widest"' },
      { name: 'as', value: 'HeadingElement', description: '"h1" | "h2" | "h3" | "h4" | "h5" | "h6"' },

    ]}
  >

  </DataTable>
