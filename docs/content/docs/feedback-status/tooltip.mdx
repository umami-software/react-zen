# Tooltip

A tooltip displays a description of an element on hover or focus.

<Example>
  <TooltipTrigger delay={0}>
    <Button variant="secondary">Hover me</Button>
    <Tooltip>Hello</Tooltip>
  </TooltipTrigger>

  ```jsx
  <TooltipTrigger>
    <Button>Hover me</Button>
    <Tooltip>Hello</Tooltip>
  </TooltipTrigger>
  ```
</Example>
