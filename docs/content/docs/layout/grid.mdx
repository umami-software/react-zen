# Grid

A CSS Grid layout component for creating two-dimensional layouts with rows and columns. Ideal for building responsive page layouts, card grids, dashboards, and any design requiring precise control over both horizontal and vertical alignment.

<Example>
  <Grid gap="3" columns="3">
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
  </Grid>

  ```jsx
  <Grid gap="3" columns="3">
    <Box>1</Box>
    <Box>2</Box>
    <Box>3</Box>
  </Grid>
  ```
</Example>

## Responsive columns

Use responsive objects to change the grid layout at different breakpoints.

<Example>
  <Grid gap="3" columns={{ base: '1', sm: '2', lg: '4' }}>
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
  </Grid>

  ```jsx
  <Grid gap="3" columns={{ base: '1', sm: '2', lg: '4' }}>
    <Box>1</Box>
    <Box>2</Box>
    <Box>3</Box>
    <Box>4</Box>
  </Grid>
  ```
</Example>

## Custom templates

For complex layouts, pass a custom CSS grid-template string.

<Example>
  <Grid gap="3" columns="200px 1fr 1fr">
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
    <ExampleBox />
  </Grid>

  ```jsx
  <Grid gap="3" columns="200px 1fr 1fr">
    <Box>Fixed</Box>
    <Box>Flex</Box>
    <Box>Flex</Box>
  </Grid>
  ```
</Example>

## Responsive custom templates

Custom CSS templates can also be responsive by passing an object with breakpoint keys.

<Example>
  <Grid gap="3" columns={{ base: '1fr', lg: 'auto 1fr' }}>
    <ExampleBox />
    <ExampleBox />
  </Grid>

  ```jsx
  <Grid gap="3" columns={{ base: '1fr', lg: 'auto 1fr' }}>
    <Box>Sidebar</Box>
    <Box>Content</Box>
  </Grid>
  ```
</Example>

## Grid item placement

<Example>
  <Grid gap="3" columns="3" rows="3">
    <ExampleBox width="auto" gridRow="1 / 2" gridColumn="1 / 3" />
    <ExampleBox width="auto" height="auto" gridRow="2 / 4" gridColumn="1 / 3" />
    <ExampleBox height="auto" gridRow="1 / 4" gridColumn="3 / 4" />
  </Grid>

  ```jsx
  <Grid gap="3" columns="3" rows="3">
    <Box gridRow="1 / 2" gridColumn="1 / 3">Header</Box>
    <Box gridRow="2 / 4" gridColumn="1 / 3">Main</Box>
    <Box gridRow="1 / 4" gridColumn="3 / 4">Sidebar</Box>
  </Grid>
  ```
</Example>
