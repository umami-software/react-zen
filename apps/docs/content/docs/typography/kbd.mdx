# Kbd

Kbd displays keyboard input such as shortcuts and key combinations.

<Example>
  <Text><Kbd>⌘</Kbd> <Kbd>K</Kbd></Text>

  ```jsx
  <Kbd>⌘</Kbd> <Kbd>K</Kbd>
  ```
</Example>

## Combinations

<Example>
  <Text>Press <Kbd>Ctrl + Shift + P</Kbd> to open the command palette.</Text>

  ```jsx
  <Text>Press <Kbd>Ctrl + Shift + P</Kbd> to open the command palette.</Text>
  ```
</Example>

## Component props

Accepts standard HTML props and renders a native `kbd` element.
