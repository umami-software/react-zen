# Aspect Ratio

Aspect ratio constrains its content to a fixed width-to-height ratio.

<Example>
  <Box width="400px">
    <AspectRatio ratio={16 / 9}>
      <img src="/images/sample.jpg" alt="Sample" style={{ borderRadius: '8px' }} />
    </AspectRatio>
  </Box>

  ```jsx
  <AspectRatio ratio={16 / 9}>
    <img src="/images/sample.jpg" alt="Sample" />
  </AspectRatio>
  ```
</Example>
