# Separator

A separator visually divides content into groups.

<Example>
  <Box width="300px">
    <Text>Above the line</Text>
    <Box paddingY="3">
      <Separator />
    </Box>
    <Text>Below the line</Text>
  </Box>

  ```jsx
  <Text>Above the line</Text>
  <Separator />
  <Text>Below the line</Text>
  ```
</Example>

## Vertical

<Example>
  <Flexbox gap="3" alignItems="center">
    <Text>Left</Text>
    <Separator orientation="vertical" style={{ height: '24px' }} />
    <Text>Right</Text>
  </Flexbox>

  ```jsx
  <Flexbox gap="3" alignItems="center">
    <Text>Left</Text>
    <Separator orientation="vertical" />
    <Text>Right</Text>
  </Flexbox>
  ```
</Example>
