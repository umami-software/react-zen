# Context Menu

A context menu opens a menu at the pointer position when a user right clicks the trigger area.
Pass two children: the trigger element followed by a `Menu`.

<Example>
  <ContextMenu>
    <Box
      border
      borderRadius="md"
      padding="8"
      style={{ borderStyle: 'dashed' }}
    >
      Right click here
    </Box>
    <Menu>
      <MenuItem>Cut</MenuItem>
      <MenuItem>Copy</MenuItem>
      <MenuItem>Paste</MenuItem>
      <MenuSeparator />
      <MenuItem>Delete</MenuItem>
    </Menu>
  </ContextMenu>

  ```jsx
  <ContextMenu>
    <Box border borderRadius="md" padding="8" style={{ borderStyle: 'dashed' }}>

      <MenuItem>Paste</MenuItem>

    </Menu>
  </ContextMenu>
  ```
</Example>
