# Breadcrumbs

Breadcrumbs display a hierarchy of links to the current page or resource in an application.

<Example>
  <Breadcrumbs>
    <Breadcrumb>
      <a href="#">Home</a>
    </Breadcrumb>
    <Breadcrumb>
      <a href="#">One</a>
    </Breadcrumb>
    <Breadcrumb>
      <a href="#">Two</a>
    </Breadcrumb>
    <Breadcrumb>Three</Breadcrumb>
  </Breadcrumbs>

  ```jsx
  <Breadcrumbs>
    <Breadcrumb><a href="#">Home</a></Breadcrumb>
    <Breadcrumb><a href="#">One</a></Breadcrumb>
    <Breadcrumb><a href="#">Two</a></Breadcrumb>
    <Breadcrumb>Three</Breadcrumb>
  </Breadcrumbs>
  ```
</Example>
