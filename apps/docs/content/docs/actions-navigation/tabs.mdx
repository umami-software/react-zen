# Tabs

Tabs organize content into multiple sections and allow users to navigate between them.

<Example>
  <Tabs>
    <TabList>
      <Tab id="one">One</Tab>
      <Tab id="two">Two</Tab>
      <Tab id="three">Three</Tab>
    </TabList>
    <TabPanel id="one">Content for tab one.</TabPanel>
    <TabPanel id="two">Content for tab two.</TabPanel>
    <TabPanel id="three">Content for tab three.</TabPanel>
  </Tabs>

  ```jsx
  <Tabs>
    <TabList>
      <Tab id="one">One</Tab>
      <Tab id="two">Two</Tab>
      <Tab id="three">Three</Tab>
    </TabList>
    <TabPanel id="one">Content for tab one.</TabPanel>
    <TabPanel id="two">Content for tab two.</TabPanel>
    <TabPanel id="three">Content for tab three.</TabPanel>
  </Tabs>
  ```
</Example>
