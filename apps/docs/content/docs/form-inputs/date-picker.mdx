# Date Picker

A date picker allows a user to select a date from a popup calendar.

<Example>
  <DatePicker label="Date" />

  ```jsx
  <DatePicker label="Date" />
  ```
</Example>

## Default value

<Example>
  <DatePicker label="Date" defaultValue={new Date()} />

  ```jsx
  <DatePicker label="Date" defaultValue={new Date()} />
  ```
</Example>

## Format options

Dates are formatted with `Intl.DateTimeFormat` using the `locale` and `formatOptions` props.

<Example>
  <DatePicker label="Date" defaultValue={new Date()} formatOptions={{ dateStyle: 'full' }} />

  ```jsx
  <DatePicker label="Date" defaultValue={new Date()} formatOptions={{ dateStyle: 'full' }} />
  ```
</Example>

## Disabled

<Example>
  <DatePicker label="Date" isDisabled />

  ```jsx
  <DatePicker label="Date" isDisabled />
  ```
</Example>

## Controlled

<Example>
  <DatePickerExample />

  ```jsx
  const [date, setDate] = useState();

  <DatePicker label="Date" value={date} onChange={setDate} />
  ```
</Example>
