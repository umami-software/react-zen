# Number Field

A number field allows a user to enter a numeric value with stepper buttons for incrementing and decrementing.

<Example>
  <Box width="200px"><NumberField label="Quantity" defaultValue={1} /></Box>

  ```jsx
  <NumberField label="Quantity" defaultValue={1} />
  ```
</Example>

## Min, max, and step

<Example>
  <Box width="200px"><NumberField label="Percent" defaultValue={50} minValue={0} maxValue={100} step={10} /></Box>

  ```jsx
  <NumberField label="Percent" defaultValue={50} minValue={0} maxValue={100} step={10} />
  ```
</Example>

## Format options

Values are formatted with `Intl.NumberFormat` using the `formatOptions` prop.

<Example>
  <Box width="200px"><NumberField label="Price" defaultValue={25} formatOptions={{ style: 'currency', currency: 'USD' }} /></Box>

  ```jsx
  <NumberField
    label="Price"
    defaultValue={25}
    formatOptions={{ style: 'currency', currency: 'USD' }}
  />
  ```
</Example>

## Disabled

<Example>
  <Box width="200px"><NumberField label="Quantity" defaultValue={1} isDisabled /></Box>

  ```jsx
  <NumberField label="Quantity" defaultValue={1} isDisabled />
  ```
</Example>
