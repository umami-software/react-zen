# Carousel

A carousel displays a set of slides that users can scroll through.

<Example>
  <Box width="400px">
    <Carousel>
      <CarouselItem>
        <Box height="200px" backgroundColor="surface-raised" border borderRadius="lg" display="flex" alignItems="center" justifyContent="center">1</Box>
      </CarouselItem>
      <CarouselItem>
        <Box height="200px" backgroundColor="surface-raised" border borderRadius="lg" display="flex" alignItems="center" justifyContent="center">2</Box>
      </CarouselItem>
      <CarouselItem>
        <Box height="200px" backgroundColor="surface-raised" border borderRadius="lg" display="flex" alignItems="center" justifyContent="center">3</Box>
      </CarouselItem>
    </Carousel>
  </Box>

  ```jsx
  <Carousel>
    <CarouselItem>
      <Box height="200px">1</Box>
    </CarouselItem>
    <CarouselItem>
      <Box height="200px">2</Box>
    </CarouselItem>
    <CarouselItem>
      <Box height="200px">3</Box>
    </CarouselItem>
  </Carousel>
  ```
</Example>

</Example>

## Component props

### Carousel

<Example>
  <DataTable
    data={[

### CarouselItem

Accepts standard HTML div props. Each item renders as a slide.
