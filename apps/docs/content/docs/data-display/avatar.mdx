# Avatar

An avatar displays a user's profile image with a fallback to their initials.

<Example>
  <Avatar src="/images/sample.jpg" alt="Jane Doe" />

  ```jsx
  <Avatar src="/images/sample.jpg" alt="Jane Doe" />
  ```
</Example>

## Sizes

<Example>
  <Avatar alt="Jane Doe" size="xs" />
  <Avatar alt="Jane Doe" size="sm" />
  <Avatar alt="Jane Doe" size="md" />
  <Avatar alt="Jane Doe" size="lg" />
  <Avatar alt="Jane Doe" size="xl" />

  ```jsx
  <Avatar alt="Jane Doe" size="xs" />
  <Avatar alt="Jane Doe" size="sm" />
  <Avatar alt="Jane Doe" size="md" />
  <Avatar alt="Jane Doe" size="lg" />
  <Avatar alt="Jane Doe" size="xl" />
  ```
</Example>

## Fallback

When no image is provided, the avatar shows the initials derived from `alt`, or a custom `fallback`.

<Example>
  <Avatar alt="Jane Doe" />
  <Avatar fallback="?" />

  ```jsx
  <Avatar alt="Jane Doe" />
  <Avatar fallback="?" />
  ```
</Example>
