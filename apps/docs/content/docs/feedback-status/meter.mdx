# Meter

A meter represents a measured value within a known range, such as disk usage or battery level.

<Example>
  <Box width="300px"><Meter label="Storage used" value={60} showValue /></Box>

  ```jsx
  <Meter label="Storage used" value={60} showValue />
  ```
</Example>

## Custom range

<Example>
  <Box width="300px"><Meter label="Battery" value={2500} minValue={0} maxValue={5000} showValue /></Box>

  ```jsx
  <Meter label="Battery" value={2500} minValue={0} maxValue={5000} showValue />
  ```
</Example>

## Format options

Values are formatted with `Intl.NumberFormat` using the `formatOptions` prop.

<Example>
  <Box width="300px"><Meter label="Memory" value={0.35} minValue={0} maxValue={1} showValue formatOptions={{ style: 'percent' }} /></Box>

  ```jsx
  <Meter
    label="Memory"
    value={0.35}
    minValue={0}
    maxValue={1}
    showValue
    formatOptions={{ style: 'percent' }}
  />
  ```
</Example>
