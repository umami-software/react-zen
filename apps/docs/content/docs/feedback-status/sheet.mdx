# Sheet

A sheet is a panel that slides in from the edge of the screen, overlaying the current content.

<Example>
  <SheetExample />

  ```jsx
  <DialogTrigger>
    <Button variant="outline">Open sheet</Button>
    <Sheet side="right">
      <SheetHeader title="Settings" />
      <Text>Sheet content slides in from the right.</Text>
    </Sheet>
  </DialogTrigger>
  ```
</Example>

## Size

The `size` prop sets the width for left and right sheets, and the height for top and bottom sheets.

<Example>
  <DialogTrigger>
    <Button variant="outline">Wide sheet</Button>
    <Sheet side="right" size="40rem">
      <SheetHeader title="Details" />
      <Text>A wider sheet panel.</Text>
    </Sheet>
  </DialogTrigger>

  ```jsx
  <DialogTrigger>
    <Button variant="outline">Wide sheet</Button>
    <Sheet side="right" size="40rem">
      <SheetHeader title="Details" />
      <Text>A wider sheet panel.</Text>
    </Sheet>
  </DialogTrigger>
  ```
</Example>
