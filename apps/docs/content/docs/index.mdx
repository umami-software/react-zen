# Introduction

**Zen** is the design system and React component collection used by [Umami](https://umami.is).
It contains the design tokens, layout primitives, and controls used across
Umami products.

This documentation is a reference for the system. It covers installation, component APIs,
and design tokens.

Source code is available on [GitHub](https://github.com/umami-software/react-zen).

## Implementation

The current components follow [shadcn](https://ui.shadcn.com/) conventions and use
[Base UI](https://base-ui.com/) primitives for interactive behavior. Styling is implemented with
[Tailwind CSS](https://tailwindcss.com/) and CSS custom properties.

## What is included

- Design tokens for [color](/docs/design/colors), [type](/docs/design/fonts),
  [spacing](/docs/design/spacing), [shadows](/docs/design/shadows), and
  [borders](/docs/design/borders)
- Layout and typography primitives
- Form controls and interactive components
- Components shared across Umami products

## Basic usage

After [installing the package](/docs/install), wrap the application with `ZenProvider` and import
the components you need.

```jsx
import { Button, ZenProvider } from '@umami/react-zen';

export function App() {
  return (
    <ZenProvider>
      <Button variant="primary">Save</Button>
    </ZenProvider>
  );
}
```
