title: Flexbox

Component for building flexible box layouts.

<Example gap="8">
  <Column alignItems="center">
    <Label>Row</Label>
    <Flexbox direction="row" gap="3">
      <ExampleBox />
      <ExampleBox />
      <ExampleBox />
    </Flexbox>
  </Column>
  <Column>
    <Label>Column</Label>
    <Flexbox direction="column" gap="3">
      <ExampleBox />
      <ExampleBox />
      <ExampleBox />
    </Flexbox>
  </Column>
</Example>

## Component props

A Flexbox renders a [Box](/docs/layout/box) as a child, so it also accepts all Box props.
